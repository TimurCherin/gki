# Starter Fastify

Створює базову структуру сайту на Fastify з необхідними функціями.

## Install

```bash
npm install opengis@starter
```

```bash
npm i # install dependencies
npm i --production # only for production
```

```bash
npm start #  debug mode

# production mode
npm run build
npm run prod 
```
